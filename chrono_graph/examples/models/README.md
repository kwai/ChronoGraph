This directoy contains some examples of using graph client.

## Install Python Client
See `chrono_graph/pypi_src/README.md`.

## Other Prerequisites
- python 3.8+ (version specified by MODULE.bazel in root dir of this project)
- pytorch
- torch_geometric
- scikit-learn
- pandas

## Run GNN model demo

#### GCN
```bash
sh gcn/run.sh
```
#### GraphSage

```bash
sh graphasage/run.sh
```
